# Introduction
Graph and dataset of connections between anion websites.

# Project objectives:

- Get an idea of ​​how many sites it is possible to discover through link scraping on onion sites.
- Assemble a relationship graph of onion sites.
- Provide a complete list of discovered sites.
- Apply machine learning techniques to categorize websites based on description and name.
- Apply ranking techniques to provide a search engine for onion sites.
